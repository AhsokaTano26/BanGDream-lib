# ※此为Bushiroad在线商店限定销售、完全受注生产商品。

BanG Dream! 推出的全新媒体混合项目「梦之结唱」

「Synthesizer V AI 梦之结唱 POPY」是使用最新AI技术开发的Synthesizer V专用歌声数据库，可实现逼真自然的人声演唱。
能够逼真地再现Poppin'Party吉他手兼主唱户山香澄的歌声。

「Synthesizer V」是由Dreamtonics株式会社开发的歌声合成软件，兼具强大的语音处理引擎和直观灵活的用户界面。
只需描绘旋律并输入歌词，即可轻松创作原创歌曲。

使用Synthesizer V系列的编辑器「Synthesizer V Studio Pro」，可支持录制语言以外的多语言演唱。例如，使用日语录制的歌声数据库可以进行英语、汉语、粤语演唱。
此外，还可使用可切换演唱风格的声乐风格功能，根据歌曲风格或场景，将音色变化为「Soft」「Powerful」「Mellow」「Joyful」四种类型。
也支持VST3和AU插件格式使用。

※本产品附带「Synthesizer V Studio Pro」。

## 运行环境

・操作系统
Windows 11/10 或更高版本（64位）
macOS 10.13 或更高版本
Linux Ubuntu 20.04 或更高版本（64位）

・CPU
Intel Core i5以上
或同等AMD处理器

AI语音高质编辑时：
推荐 Intel Core 系列第四代i5（i5-4xxx）以上
AMD Athlon X4 845以上，包括Ryzen系列

・内存
2GB以上

・硬盘容量
1GB以上
※根据安装的歌声数据库数量需要相应容量。

・显示器分辨率
1280×800以上分辨率

・其他
必须配备音频设备、互联网连接环境

※计算机需连接互联网，以便进行激活（注册）、最新版本更新以及接受有关软件的咨询等用户支持。
※即使满足上述运行条件，也不保证在所有计算机上均可运行。
※根据计算机的综合性能，可同时使用的音轨数等性能会有所不同。
※安装歌声数据库前，需先安装Synthesizer V Studio。
