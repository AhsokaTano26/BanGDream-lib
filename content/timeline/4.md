本次更新一次性上线 7 项新功能，覆盖搜索、内容发现、离线访问和视觉体验。

### 全站搜索

- 按 ⌘K / Ctrl+K 打开搜索模态框，支持模糊搜索 2900+ 篇内容。
- 搜索结果按集合类型分组显示（Live/Event、News、Discographies 等），支持键盘导航。
- 桌面端侧边栏顶部、移动端右上角均有搜索入口。

### 历史上的今天

- 首页日历下方新增「历史上的今天」模块，自动展示今天在 BanG Dream! 历史上的事件和生日。
- 独立页面 `/on-this-day` 展示完整记录，按年份倒序分组，含角色/声优生日卡片。
- 侧边栏新增导航链接。

### 活动倒计时

- 右侧栏顶部新增实时倒计时组件，显示距离最近 Live/Event 的天、时、分、秒。
- 使用 requestAnimationFrame 驱动，点击可跳转活动详情。

### 唱片数据库增强

- Discographies 页面新增乐队筛选标签、标题搜索框、升/降序排列切换。
- 页面顶部展示各乐队发行数量条形图统计。

### PWA 离线支持

- 集成 @vite-pwa/nuxt，支持安装到手机桌面。
- Service Worker 缓存静态资源和 Google Fonts，离线时仍可浏览已访问页面。

### 深色/浅色主题切换

- 侧边栏底部新增主题切换按钮（太阳/月亮图标）。
- 浅色模式使用 CSS 变量覆盖，设置自动保存到 localStorage。

### 关联推荐

- 每篇文章详情页底部自动推荐 4 条相关内容。
- 基于同乐队 > 同类型 > 时间相近的评分算法智能匹配。

### 修复

- 爬虫 build_events 增加回退抓取逻辑：当 WordPress API content.rendered 为空时，抓取实际页面提取内容。
- CI 管道增加空正文文件自动检测与修复。
- extractShowTime 时间正则支持简繁体中文。
