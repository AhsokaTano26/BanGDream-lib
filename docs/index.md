!!! abstract "项目简介"
**BanG Dream!** 是一个以“二次元 x 三次元”为特色的多媒体音乐项目。

    它包含了动画、手游（《BanG Dream! Girls Band Party!》）、漫画，以及最重要的——**声优们亲自演奏的 Live 演唱会！**
    
    这个网站致力于为你呈现这个充满音乐和羁绊的世界的**核心信息**和**最新动态**。
